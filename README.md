# SOLARWEATHER Web App

## Overview
**SOLARWEATHER** is a web-based application designed to provide interactive information about solar and space weather.  
It allows users to explore solar phenomena such as solar flares, sunspots, and geomagnetic activity through an intuitive browser interface.  

This project was developed as a **personal innovation project** to increase awareness about space weather and its effects on Earth.

---

## Features
- **Interactive web interface:** Built using HTML, CSS, and JavaScript for a smooth, responsive experience.  
- **Solar data exploration:** Users can view and analyze solar activity information in a user-friendly layout.  
- **Dynamic content:** JavaScript is used to display live or simulated solar weather events.  
- **Educational purpose:** Helps students and enthusiasts understand space phenomena visually.
